🧠 Dynamic Internal Linking (AI Automation)

Built an AI-powered multilingual internal linking system that revolutionized how users discover related products on Trendyol. The system uses advanced LLM-based automation to dynamically suggest relevant product pages based on user behavior and semantic relevance.

For example, when a user lands on an iPhone product page, the system automatically surfaces contextually relevant links to compatible accessories like AirPods, phone cases, or charging cables. This intelligent linking improved both user experience and SEO performance.

Technical Achievement: The system processes thousands of URLs across 15 languages, optimizing crawl efficiency and improving semantic relevance. It resulted in a 40% increase in internal page views and improved overall site engagement metrics.

🧩 Dynamic Rendering as a Service & SSR Rendering

A scalable rendering solution that transformed how Trendyol handled content visibility for Googlebot. By converting JavaScript-heavy pages into static HTML versions through server-side rendering, this project significantly improved page load speeds and enhanced critical SEO fragment visibility.

This infrastructure was developed as a service, enabling over 100 internal services to be easily managed. It reduced international launch timelines from 6 months to just 2 days and improved Googlebot’s page loading speed by 80%.

Key Impact: Delivered an SEO-compliant rendering platform that accelerated country launches and improved search engine indexing quality at scale.

Product Management SEO Infrastructure SSR Scalability Performance Optimization

🧩 SEO Crawler Project

An internal tool designed to scan websites, assess vital meta tags, and identify SEO fragments. Integrated with Google Search Console APIs, the crawler delivers advanced diagnostics including page scoring, Web Vitals metrics, and content optimization opportunities.

The crawler includes an AI assistant that generates SEO content (title, description) using LLMs, recommends page improvements, sends auto-alerts, and can fix minor issues when granted access to the site’s repository.

Key Impact: Streamlined on-site SEO auditing and reduced manual intervention by offering AI-powered diagnostics and content creation.
